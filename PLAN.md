## Næste fokus

1. Fasthold Paper 26 og Java 25 som teknisk baseline i repo-dokumentationen.
2. Færdiggør BlueMap world render, så kortet kan bruges som spiller- og adminværktøj.
3. Afklar landgrænse-systemet, før WorldGuard/WorldEdit-fundamentet bygges ud.
4. Byg Discord-integrationerne efter serverstatus, player count og rolle-sync er defineret.
5. Hold Notion-roadmap og GitHub-plan synkroniseret ved større ændringer.
