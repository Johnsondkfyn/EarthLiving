# Earth Living Discord Foundation

Updated: 2026-05-18

## Completed

- Server renamed and branded as Earth Living.
- Earth Living server icon added.
- Old channels were removed where Discord allowed it.
- Discord-protected community channels were preserved and renamed where needed.
- Main channel structure was rebuilt around the Earth Living project.
- Welcome, rules, announcements, server info, BlueMap and roadmap messages were added.

- `staff-updates`
- `START HERE`: rules, announcements, welcome
- `EARTH LIVING`: server-info, roadmap, bluemap, development-log
- `COMMUNITY`: general, screenshots, ideas
- `SUPPORT`: help, bug-reports
- `VOICE`: Earth Living Voice, Build Planning, AFK
- `STAFF`: staff-chat, moderation-log
