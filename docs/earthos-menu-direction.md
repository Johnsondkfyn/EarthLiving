# EarthOS Menu Direction

EarthOS should move toward a branded EarthLiving command-center interface.

## Chosen direction

Use the ChatGPT concepts as visual direction, especially:

- Command Center layout for admin/player utility.
- Futuristic hologram style for the long-term premium look.
- Dark carbon/black base with gold and cyan highlights.
- Globe/compass identity as the central EarthOS symbol.
- Menu categories that feel like an MMO world dashboard, not a normal settings screen.

## Practical implementation path

### Phase 1 - Current Minecraft inventory GUI

Use normal inventory menus because they are stable and fast to test.

Current main apps:

- World Map
- Server Events
- Passport
- Wallet
- Reports
- Server Status
- Settings

First real app:

- Reports app with category selection.
- Quick reports save player, UUID, world, coordinates, category, timestamp and status.

### Phase 2 - Better inventory styling

Improve the inventory menu with:

- Better item choices.
- Consistent lore text.
- Back/close buttons.
- Status counters.
- Admin-only menu entries.
- Menu sounds.

### Phase 3 - Resource pack UI

Add a resource pack so EarthOS can look closer to the concept images:

- Custom EarthOS hotbar icon.
- Custom item textures for apps.
- Custom font/GUI textures if needed.
- Gold/cyan EarthLiving visual identity.

## Design rule

The menu should stay useful first. Visual polish should support faster navigation, not hide the core gameplay tools.
